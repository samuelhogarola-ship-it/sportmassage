# Sports Massage Blog

Tema base de WordPress para blog, inspirado en la identidad visual de la web actual.

## Uso

1. Sube el `.zip` desde `Apariencia > Temas > Añadir nuevo > Subir tema`
2. Activa el tema
3. Configura logo, nombre del sitio y menú principal
4. Crea una página para el blog si quieres usar portada estática

## Incluye

- portada visual
- listado de artículos
- artículo individual
- páginas estáticas
- archivo
- sidebar
